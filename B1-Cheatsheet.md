# B1 Dutch Cheatsheet (Spiekbriefje)

## Quick Reference for B1 Level

---

## Perfect Tenses

### Present Perfect (VTT)
**Hebben/Zijn + voltooid deelwoord**

| Hebben | Zijn (movement/change) |
|--------|------------------------|
| Ik heb gewerkt | Ik ben gegaan |
| Ik heb gegeten | Ik ben gekomen |
| Ik heb gemaakt | Ik ben geworden |

### Past Perfect (VVT)
**Had(den) / Was/Waren + voltooid deelwoord**
- Ik **had** al **gegeten** voordat...
- Hij **was** al **vertrokken** toen...

### Strong Verb Patterns
| Pattern | Infinitive → Past → Participle |
|---------|-------------------------------|
| ij-ee-e | schrijven → schreef → geschreven |
| ie-oo-o | schieten → schoot → geschoten |
| i-o-o | drinken → dronk → gedronken |

## Passive Voice

### Formation
| Tense | Structure | Example |
|-------|-----------|---------|
| Present | word(t/en) + vd | Het **wordt gemaakt** |
| Past | werd(en) + vd | Het **werd gemaakt** |
| Perfect | is/zijn + vd | Het **is gemaakt** |

### Agent
**Door** = by (introduces who does the action)
- Het boek wordt door hem **geschreven**.

---

## Conjunctions

### Coordinating (V2 word order)
**en, of, maar, want, dus**

### Subordinating (verb → end)
| Dutch | English |
|-------|---------|
| dat | that |
| omdat | because |
| als/wanneer | when/if |
| toen | when (past) |
| voordat/nadat | before/after |
| hoewel | although |
| zodat | so that |
| terwijl | while |

### Structure
Main: Ik weet **dat** hij morgen **komt**.  
SOV in subordinate clause!

## Conditional Sentences

| Type | If-clause | Main clause |
|------|-----------|-------------|
| 0 (fact) | Present | Present |
| 1 (real) | Present | Present/Future |
| 2 (unreal) | Past | zou(den) + inf |
| 3 (past unreal) | VVT | zou(den) + hebben/zijn + vd |

### Examples
- Type 2: **Als** ik rijk **was**, **zou** ik reizen.
- Type 3: **Als** ik dat **had geweten**, **zou** ik **zijn gekomen**.

## Relative Clauses

### Die vs Dat
| Use | When |
|-----|------|
| **Die** | De-words, plurals |
| **Dat** | Het-words (singular) |

### With Prepositions
waar + preposition
- Het boek **waarover** ik sprak...
- De pen **waarmee** ik schrijf...

## Comparatives & Superlatives

### Regular
| Type | Formation |
|------|-----------|
| Comparative | + **-er** (groot → groter) |
| Superlative | + **-st** (groot → grootst) |

### Irregular
| Positive | Comparative | Superlative |
|----------|-------------|-------------|
| goed | beter | best |
| veel | meer | meest |
| weinig | minder | minst |
| graag | liever | liefst |

### Structures
- even... als = as... as
- niet zo... als = not as... as
- hoe... hoe... = the... the...

## Separable Verbs

### Split in:
- Main clause: Ik **sta** vroeg **op**.
- Imperative: **Sta op**!

### Don't split in:
- Infinitive: Ik wil **opstaan**.
- Subordinate: ...omdat ik vroeg **opsta**.
- Perfect (ge- in middle): Ik ben **opgestaan**.

### Inseparable prefixes (no ge-)
**be-, ver-, ont-, ge-, her-, er-**

---

## Future Tense

### Options
1. **Present + time word** (most common)
   - Ik kom morgen.
2. **Zullen + infinitive** (formal/emphatic)
   - Ik zal komen.
3. **Gaan + infinitive** (going to)
   - Ik ga verhuizen.

### Zou/Zouden = would
- Ik **zou** graag willen...
- **Zou** je me kunnen helpen?

## The 5 Uses of "Er"

| Type | Example |
|------|---------|
| 1. Locative (there) | Ik ben **er** geweest. |
| 2. Existential (there is) | **Er** zijn veel mensen. |
| 3. Prepositional (er+prep) | Ik denk **eraan**. |
| 4. Partitive (of them) | Ik heb **er** drie. |
| 5. Passive (dummy) | **Er** wordt gedanst. |

### Prepositional Er
| Prep | Er + prep |
|------|-----------|
| aan | eraan |
| op | erop |
| over | erover |
| met | er**mee** |
| voor | ervoor |

## Common Mistakes to Avoid

❌ Using zullen too much → Use present + time  
❌ Die/dat confusion → Check de/het  
❌ Forgetting verb at end in subordinate  
❌ Wrong separable verb position  
❌ Er + preposition for people (use hem/haar)  
❌ Mixing formal/informal in same text  

---

## B1 Exam Tips

1. **Reading:** Look for connectors to follow argument
2. **Listening:** Note time/place info, main topic
3. **Writing:** Use varied sentence structures
4. **Speaking:** Use opinion phrases, give reasons
5. **Know:** Perfect tenses, passive, subordinate clauses
6. **Practice:** Formal and informal writing

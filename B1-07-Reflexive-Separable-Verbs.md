# B1 Dutch: Reflexive and Separable Verbs (Wederkerend en Scheidbaar)

## Overview
This lesson covers two important verb types: reflexive verbs (used with reflexive pronouns) and separable verbs (which split in certain contexts).

---

## Part 1: Reflexive Verbs (Wederkerende Werkwoorden)

### What are Reflexive Verbs?
Reflexive verbs require a reflexive pronoun that refers back to the subject. The action "reflects" back on the doer.

### Reflexive Pronouns

| Subject | Reflexive Pronoun |
|---------|------------------|
| ik | me / mij / mezelf |
| jij | je / jezelf |
| u | u / zich / uzelf |
| hij | zich / zichzelf |
| zij | zich / zichzelf |
| het | zich / zichzelf |
| wij | ons / onszelf |
| jullie | je / jezelf |
| zij (they) | zich / zichzelf |

**Note:** 
- Use short forms (me, je, zich) normally
- Use long forms (mij, mezelf, zichzelf) for emphasis

---

### Types of Reflexive Verbs

#### 1. True Reflexive Verbs (Always reflexive)
These verbs ALWAYS need a reflexive pronoun:

| Dutch | English | Example |
|-------|---------|---------|
| zich vergissen | to be mistaken | Ik **vergis me** nooit. |
| zich schamen | to be ashamed | Hij **schaamt zich**. |
| zich gedragen | to behave | **Gedraag je**! |
| zich bemoeien met | to interfere | **Bemoei je** niet met mij! |
| zich herinneren | to remember | Ik **herinner me** het niet. |
| zich realiseren | to realize | Zij **realiseert zich**... |
| zich verzetten tegen | to resist | Wij **verzetten ons** tegen... |
| zich afvragen | to wonder | Ik **vraag me af**... |
| zich bevinden | to be located | Het hotel **bevindt zich**... |

#### 2. Optional Reflexive Verbs
These can be used with or without reflexive pronouns:

| Without Reflexive | With Reflexive |
|-------------------|----------------|
| Ik was (I wash) | Ik was **me** (I wash myself) |
| Ik kleed (I dress) | Ik kleed **me** aan (I dress myself) |
| Hij voelt goed (It feels good) | Hij voelt **zich** goed (He feels good) |

#### 3. Reciprocal Verbs (Each other)
When the action is mutual, use "elkaar":

| Dutch | English |
|-------|---------|
| Wij zien **elkaar** vaak. | We see each other often. |
| Zij helpen **elkaar**. | They help each other. |
| Kennen jullie **elkaar**? | Do you know each other? |

---

### Word Order with Reflexive Verbs

**Position of reflexive pronoun:**

1. **Main clause:** After the conjugated verb
   - Ik **was me** elke ochtend.
   - **Vergist** hij **zich** nooit?

2. **After inversion:** After the subject
   - Morgen **was** ik **me** niet.
   - Vandaag **voelt** hij **zich** beter.

3. **Subordinate clause:** After the subject
   - ...omdat ik **me** goed **voel**.
   - ...dat hij **zich** vergist **heeft**.

## Part 2: Separable Verbs (Scheidbare Werkwoorden)

### What are Separable Verbs?
Separable verbs have a prefix that separates from the main verb in certain situations:
- **opstaan** (to get up) → Ik **sta** om 7 uur **op**.

### Common Separable Prefixes

| Prefix | Meaning | Examples |
|--------|---------|----------|
| aan- | on, to | aankomen, aandoen, aankijken |
| af- | off, down | afmaken, afsluiten, afhalen |
| bij- | with, near | bijdragen, bijkomen |
| in- | in | indelen, invallen, intrekken |
| mee- | along | meekomen, meenemen, meedoen |
| na- | after | nadenken, nakijken |
| op- | up | opstaan, opbellen, opruimen |
| terug- | back | terugkomen, teruggeven |
| uit- | out | uitgaan, uitleggen, uitnodigen |
| voor- | before, forward | voorstellen, voorkomen |
| door- | through | doorgaan, doorlezen |

---

### When Do Separable Verbs Split?

#### They SPLIT in:

**1. Present/Past Main Clause (verb position 2)**
- Ik **sta** elke dag vroeg **op**.
- Hij **belde** mij gisteren **op**.
- **Kom** je morgen **mee**?

**2. Imperative**
- **Sta op**! (Get up!)
- **Kom binnen**! (Come in!)
- **Ga weg**! (Go away!)

#### They DON'T SPLIT in:

**1. Infinitive**
- Ik wil vroeg **opstaan**.
- Hij moet haar **opbellen**.
- Ik ga morgen **meekomen**.

**2. Subordinate Clause**
- ...omdat ik vroeg **opsta**.
- ...dat hij haar **opbelde**.
- ...wanneer je **meekomt**.

**3. Perfect Tense (ge- goes in the middle)**
- Ik heb hem **opgebeld**.
- Hij is vroeg **opgestaan**.
- Wij hebben de kamer **opgeruimd**.

---

### Common Separable Verbs

| Infinitive | Meaning | Present | Perfect |
|------------|---------|---------|---------|
| opstaan | to get up | Ik sta op | Ik ben opgestaan |
| opbellen | to call | Ik bel op | Ik heb opgebeld |
| aankomen | to arrive | Ik kom aan | Ik ben aangekomen |
| weggaan | to leave | Ik ga weg | Ik ben weggegaan |
| terugkomen | to come back | Ik kom terug | Ik ben teruggekomen |
| meekomen | to come along | Ik kom mee | Ik ben meegekomen |
| uitgaan | to go out | Ik ga uit | Ik ben uitgegaan |
| aandoen | to put on | Ik doe aan | Ik heb aangedaan |
| uitdoen | to take off | Ik doe uit | Ik heb uitgedaan |
| afmaken | to finish | Ik maak af | Ik heb afgemaakt |
| opruimen | to tidy up | Ik ruim op | Ik heb opgeruimd |
| afspreken | to arrange | Ik spreek af | Ik heb afgesproken |
| doorgaan | to continue | Ik ga door | Ik ben doorgegaan |
| uitleggen | to explain | Ik leg uit | Ik heb uitgelegd |
| meenemen | to take along | Ik neem mee | Ik heb meegenomen |

---

### Inseparable Verbs (Niet-scheidbaar)

Some prefixes NEVER separate:

| Prefix | Examples |
|--------|----------|
| be- | betalen, begrijpen, bezoeken |
| er- | erkennen |
| ge- | geloven |
| her- | herhalen, herkennen |
| ont- | ontmoeten, ontvangen, ontsnappen |
| ver- | verkopen, vertellen, vergeten |

**Note:** No "ge-" in past participle:
- betalen → betaald (NOT gebetaald)
- vergeten → vergeten
- begrijpen → begrepen

---

### Separable vs Inseparable

Some verbs can be both, with different meanings:

| Separable | Inseparable |
|-----------|-------------|
| **door**lezen (to continue reading) | door**lez**en (to read through) |
| Ik lees het boek **door**. | Ik doorlees het rapport. |
| **om**rijden (to make a detour) | om**rijd**en (to knock down) |
| **voor**komen (to happen) | voor**kom**en (to prevent) |

---

## Combined: Reflexive + Separable

Some verbs are both reflexive AND separable:

| Infinitive | Meaning | Example |
|------------|---------|---------|
| zich aankleden | to get dressed | Ik kleed me aan. |
| zich aanmelden | to register | Ik meld me aan. |
| zich omdraaien | to turn around | Hij draaide zich om. |
| zich voorstellen | to introduce oneself | Ik stel me voor. |
| zich afvragen | to wonder | Ik vraag me af... |

**Word order:**
- Main clause: Ik **kleed me** snel **aan**.
- Subordinate: ...omdat ik **me** snel **aankleed**.
- Perfect: Ik **heb me** snel **aangekleed**.

## Practice Exercises

### Exercise 1: Fill in the reflexive pronoun
1. Ik voel _______ vandaag niet lekker.
2. Hij herinnert _______ het niet meer.
3. Wij interesseren _______ voor geschiedenis.
4. Gedraag _______! (to a child)
5. Zij schamen _______ voor hun gedrag.

### Exercise 2: Separate or not? Rewrite correctly
1. Ik opstaan om 7 uur.
2. Hij moet het werk afmaken.
3. Omdat ik vroeg opsta, ...
4. Zij heeft haar moeder opgebeld.
5. Aandoen je jas!

### Exercise 3: Present to Perfect
1. Ik sta vroeg op. → Ik _______ vroeg _______.
2. Zij meldt zich aan. → Zij _______ zich _______.
3. Wij komen morgen aan. → Wij _______ morgen _______.
4. Hij maakt het werk af. → Hij _______ het werk _______.
5. Ik vergeet het nooit. → Ik _______ het nooit _______.

### Answers

**Exercise 1:**
1. me
2. zich
3. ons
4. je
5. zich

**Exercise 2:**
1. Ik **sta** om 7 uur **op**.
2. Correct (infinitive doesn't split)
3. Correct (subordinate doesn't split)
4. Correct (perfect: ge- in middle)
5. **Doe** je jas **aan**!

**Exercise 3:**
1. Ik **ben** vroeg **opgestaan**.
2. Zij **heeft** zich **aangemeld**.
3. Wij **zijn** morgen **aangekomen**.
4. Hij **heeft** het werk **afgemaakt**.
5. Ik **heb** het nooit **vergeten**. (inseparable)

## Key Points to Remember

### Reflexive Verbs
✅ Use **zich** for hij/zij/het/zij (plural)  
✅ Use **me/ons** for ik/wij  
✅ Position: after conjugated verb, after subject in subordinate  
✅ Some verbs are ALWAYS reflexive  

### Separable Verbs
✅ Split in **main clauses** and **imperatives**  
✅ Don't split in **infinitive** and **subordinate clauses**  
✅ Perfect: **ge-** goes **IN THE MIDDLE** (opgestaan)  
✅ **be-, ver-, ont-, ge-, her-** = inseparable (no ge- in perfect)

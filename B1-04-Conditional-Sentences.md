# B1 Dutch: Conditional Sentences (Conditionele Zinnen)

## Overview
Conditional sentences express situations that depend on certain conditions. Dutch uses different structures for real/possible conditions and unreal/hypothetical conditions.

---

## Types of Conditionals

| Type | Condition | Example |
|------|-----------|---------|
| Type 0 | General truth | Als water 100°C bereikt, kookt het. |
| Type 1 | Real/Possible | Als het regent, neem ik een paraplu. |
| Type 2 | Unreal present | Als ik rijk was, zou ik een huis kopen. |
| Type 3 | Unreal past | Als ik dat had geweten, was ik niet gegaan. |

---

## Type 0: General Truth (Zero Conditional)

### Structure
**Als + present tense, present tense**

Describes universal truths, scientific facts, or things that always happen.

### Examples
- **Als** je water verwarmt, **kookt** het bij 100 graden.
- **Als** je niet eet, **krijg** je honger.
- **Als** de zon schijnt, **is** het warm.
- **Als** je te hard werkt, **word** je moe.

---

## Type 1: Real Conditional (First Conditional)

### Structure
**Als + present tense, present/future tense**

Describes possible or likely situations in the present or future.

### Examples
- **Als** het morgen regent, **blijf** ik thuis.
- **Als** ik tijd heb, **kom** ik langs.
- **Als** je hard studeert, **zul** je slagen.
- **Als** hij niet belt, **ga** ik zonder hem.

### With Modal Verbs
- Als je wilt, **kun** je meekomen.
- Als het nodig is, **moet** je harder werken.
- Als je mag, **mag** je eerder naar huis.

### With Future (zullen)
- Als je komt, **zal** ik koken.
- Als het lukt, **zullen** we het vieren.

---

## Type 2: Unreal Present (Second Conditional)

### Structure
**Als + past tense (imperfectum), zou(den) + infinitive**

Describes unreal, imaginary, or unlikely situations in the present or future.

### Examples
- **Als** ik rijk **was**, **zou** ik een groot huis **kopen**.
- **Als** ik jou **was**, **zou** ik dat niet **doen**.
- **Als** hij hier **woonde**, **zouden** we vaker **afspreken**.
- **Als** ik het **wist**, **zou** ik het je **vertellen**.

### With Modal Verbs in zou-clause
- Als ik kon, **zou** ik je **helpen**.
- Als het mocht, **zou** ik eerder **vertrekken**.
- Als hij wilde, **zou** hij het **kunnen**.

### Alternative: Using Past Tense in Both Clauses
More informal/spoken:
- **Als** ik rijk **was**, **kocht** ik een groot huis.
- **Als** ik jou **was**, **deed** ik dat niet.

---

## Type 3: Unreal Past (Third Conditional)

### Structure
**Als + past perfect (had + participle), zou(den) + hebben/zijn + participle**

OR

**Als + past perfect, past perfect (had + participle)**

Describes hypothetical situations in the past that didn't happen.

### Examples with zou + perfect
- **Als** ik dat **had geweten**, **zou** ik niet **zijn gegaan**.
- **Als** hij harder **had gewerkt**, **zou** hij **zijn geslaagd**.
- **Als** we eerder **waren vertrokken**, **zouden** we de trein **hebben gehaald**.

### Alternative: Using Past Perfect in Both Clauses
- **Als** ik dat **had geweten**, **was** ik niet **gegaan**.
- **Als** hij harder **had gewerkt**, **was** hij **geslaagd**.
- **Als** we eerder **waren vertrokken**, **hadden** we de trein **gehaald**.

---

## Zou/Zouden - Conjugation

| Pronoun | Conjugation |
|---------|-------------|
| ik | zou |
| jij | zou |
| u | zou |
| hij/zij/het | zou |
| wij | zouden |
| jullie | zouden |
| zij | zouden |

### Uses of Zou/Zouden
1. **Conditional** (would): Als ik kon, zou ik helpen.
2. **Polite requests**: Zou je me kunnen helpen?
3. **Suggestions**: Zouden we naar de film gaan?
4. **Uncertainty**: Hij zou ziek zijn. (He is supposedly sick.)

---

## Word Order with Als

### Als-clause First (Inversion in Main Clause)
- **Als het regent**, **blijf ik** thuis.
- **Als ik rijk was**, **zou ik** reizen.

### Als-clause Second (Normal Order)
- Ik **blijf** thuis **als het regent**.
- Ik **zou** reizen **als ik rijk was**.

---

## Other Conditional Words

### Mits (Provided that)
- Je mag gaan, **mits** je je huiswerk af **hebt**.
- Ik help je, **mits** je het ook probeert.

### Tenzij (Unless)
- Ik kom, **tenzij** het **regent**.
- We gaan door, **tenzij** er bezwaren **zijn**.
- Hij slaagt, **tenzij** hij stopt met studeren.

### Indien (If - formal)
- **Indien** u vragen **heeft**, neem contact met ons op.
- **Indien** mogelijk, verzoeken wij u...

### Stel dat / Gesteld dat (Suppose that)
- **Stel dat** je morgen niet kunt komen, wat dan?
- **Gesteld dat** hij gelijk heeft, wat doen we dan?

### Mocht (In case/If - uncertain)
- **Mocht** je hem **zien**, zeg dan dat ik belde.
- **Mocht** dat **lukken**, dan is het mooi.
- **Mochten** er problemen **zijn**, bel ons.

---

## Conditional Without Als

### Inversion as Condition (Formal)
Instead of "als", you can use inversion:
- **Had** ik dat geweten, dan was ik niet gegaan. (= Als ik dat had geweten...)
- **Was** ik rijk, dan zou ik reizen. (= Als ik rijk was...)
- **Kun** je komen, bel me. (= Als je kunt komen...)

This is more formal and literary.

---

## Mixed Conditionals

Sometimes the time reference differs between clauses:

### Past Condition → Present Result
- **Als** ik harder **had gestudeerd** (past), **was** ik nu dokter (present).
- **Als** zij niet **was verhuisd** (past), **woonden** we nu nog samen.

### Present Condition → Past Result
- **Als** hij zo slim **was** (present), **had** hij de test wel **gehaald** (past).

## Common Expressions with Zou

| Dutch | English |
|-------|---------|
| Zou je...? | Would you...? |
| Zou het kunnen dat...? | Could it be that...? |
| Dat zou kunnen. | That could be. |
| Ik zou zeggen... | I would say... |
| Ik zou willen... | I would like... |
| Zou je zo vriendelijk willen zijn om...? | Would you be so kind as to...? |
| Het zou me niet verbazen als... | It wouldn't surprise me if... |
| Ik zou niet weten... | I wouldn't know... |

### Exercise 1: Identify the conditional type (0, 1, 2, or 3)
1. Als je hard werkt, verdien je meer geld.
2. Als ik rijk was, zou ik een boot kopen.
3. Als water bevriest, wordt het ijs.
4. Als hij had gestudeerd, was hij geslaagd.
5. Als het morgen mooi weer is, gaan we naar het strand.

### Exercise 2: Complete with the correct form
1. Als ik jou _______ (zijn), zou ik dat niet doen.
2. Als hij harder _______ (werken), zou hij meer verdienen.
3. Als ik dat _______ (weten), had ik je gebeld. (type 3)
4. Als het _______ (regenen), neem ik een paraplu. (type 1)
5. Als we eerder _______ (vertrekken), hadden we de trein gehaald.

### Exercise 3: Translate to Dutch
1. If I were you, I would stay home.
2. If it rains tomorrow, we will cancel the picnic.
3. If he had called, I would have answered.
4. I would help you if I could.
5. Unless it rains, we will go to the park.

### Answers

**Exercise 1:**
1. Type 1 (real/possible)
2. Type 2 (unreal present)
3. Type 0 (general truth)
4. Type 3 (unreal past)
5. Type 1 (real/possible)

**Exercise 2:**
1. was
2. werkte

4. regent

**Exercise 3:**

2. Als het morgen regent, annuleren we de picknick.
3. Als hij had gebeld, had ik opgenomen / zou ik hebben opgenomen.
4. Ik zou je helpen als ik kon.
5. Tenzij het regent, gaan we naar het park.

## Key Points to Remember

✅ **Type 0**: General truth - present + present  
✅ **Type 1**: Possible - present + present/future  
✅ **Type 2**: Unreal present - past + zou + infinitive  
✅ **Type 3**: Unreal past - past perfect + zou + perfect / past perfect  
✅ **Als-first**: causes inversion in main clause  
✅ **Zou/zouden**: conditional would  
✅ **Mits**: provided that  
✅ **Tenzij**: unless

# A2 Dutch: Pronouns (Voornaamwoorden)

## Overview
Dutch has several types of pronouns. Understanding them is essential for natural communication.

---

## Personal Pronouns (Persoonlijke Voornaamwoorden)

### Subject Pronouns
| Dutch (Stressed) | Dutch (Unstressed) | English |
|------------------|-------------------|---------|
| ik | 'k | I |
| jij | je | you (informal) |
| u | - | you (formal) |
| hij | ie | he |
| zij | ze | she |
| het | 't | it |
| wij | we | we |
| jullie | - | you (plural) |
| zij | ze | they |

### Object Pronouns (Direct Object)
| Subject | Object | English |
|---------|--------|---------|
| ik | mij / me | me |
| jij | jou / je | you |
| u | u | you (formal) |
| hij | hem / 'm | him |
| zij | haar / ze / d'r | her |
| het | het / 't | it |
| wij | ons | us |
| jullie | jullie | you (plural) |
| zij | hen / hun / ze | them |

### Examples
- Ik zie **hem**. (I see him.)
- Kun je **mij** helpen? (Can you help me?)
- Ik geef **haar** het boek. (I give her the book.)
- Ken je **ons**? (Do you know us?)
- Ik bel **je** morgen. (I'll call you tomorrow.)

## Possessive Pronouns (Bezittelijke Voornaamwoorden)

### Before Nouns (Determiners)
| Pronoun | Possessive | English |
|---------|------------|---------|
| ik | mijn | my |
| jij | jouw / je | your |
| u | uw | your (formal) |
| hij | zijn | his |
| zij | haar | her |
| het | zijn | its |
| wij | ons / onze | our |
| jullie | jullie | your (plural) |
| zij | hun | their |

### Ons vs Onze
- **Ons** + het-words: ons huis, ons kind
- **Onze** + de-words and plurals: onze auto, onze kinderen

### Examples
- Dit is **mijn** boek. (This is my book.)
- Waar is **jouw** auto? (Where is your car?)
- **Zijn** moeder is aardig. (His mother is nice.)
- **Onze** kinderen zijn groot. (Our children are big.)
- **Hun** huis is mooi. (Their house is beautiful.)

### Independent Possessives (Standing Alone)
| Possessive | Independent Form |
|------------|-----------------|
| mijn | die/dat van mij |
| jouw | die/dat van jou |
| zijn | die/dat van hem |
| haar | die/dat van haar |
| ons/onze | die/dat van ons |
| hun | die/dat van hen |

**Example:**
- Wiens boek is dit? Dat is **die van mij**. (Whose book is this? That's mine.)

## Reflexive Pronouns (Wederkerende Voornaamwoorden)

| Subject | Reflexive | English |
|---------|-----------|---------|
| ik | me / mij / mezelf | myself |
| jij | je / jezelf | yourself |
| u | u / zich / uzelf | yourself |
| hij | zich / zichzelf | himself |
| zij | zich / zichzelf | herself |
| het | zich / zichzelf | itself |
| wij | ons / onszelf | ourselves |
| jullie | je / jezelf | yourselves |
| zij | zich / zichzelf | themselves |

### Examples
- Ik was **me**. (I wash myself.)
- Hij schaamt **zich**. (He is ashamed.)
- Wij vermaken **ons**. (We enjoy ourselves.)
- Gedraag **je**! (Behave yourself!)

## Demonstrative Pronouns (Aanwijzende Voornaamwoorden)

### This/That
| | Near (this) | Far (that) |
|---|------------|-----------|
| de-words | deze | die |
| het-words | dit | dat |
| Plural | deze | die |

### Examples
- **Dit** boek is interessant. (This book is interesting.)
- **Die** auto is duur. (That car is expensive.)
- **Deze** schoenen zijn mooi. (These shoes are beautiful.)
- **Dat** huis is groot. (That house is big.)

### Standing Alone
- Wat is **dit**? (What is this?)
- Wie is **dat**? (Who is that?)
- **Deze** is beter dan **die**. (This one is better than that one.)

---

## Relative Pronouns (Betrekkelijke Voornaamwoorden)

### Basic Forms
| | de-words | het-words |
|---|----------|-----------|
| Subject | die | dat |
| Object | die | dat |
| Possessive | wiens/van wie | waarvan |

### Examples
- De man **die** daar staat, is mijn vader. (The man who stands there is my father.)
- Het boek **dat** ik lees, is goed. (The book that I'm reading is good.)
- De vrouw **van wie** ik het heb gehoord... (The woman from whom I heard it...)

### With Prepositions
For things, use **waar + preposition**:
- Het huis **waarin** ik woon... (The house in which I live...)
- De stoel **waarop** ik zit... (The chair on which I sit...)

### Common Indefinite Pronouns
| Dutch | English |
|-------|---------|
| iemand | someone |
| niemand | nobody |
| iedereen | everyone |
| iets | something |
| niets | nothing |
| alles | everything |
| veel | much/many |

| sommige | some |
| andere | other |
| elk/elke | each/every |
| alle | all |
| beide | both |

### Examples
- Is er **iemand** thuis? (Is someone home?)
- Ik heb **niets** gezien. (I saw nothing.)
- **Iedereen** is welkom. (Everyone is welcome.)
- **Sommige** mensen denken... (Some people think...)
- Heb je **iets** nodig? (Do you need something?)

## Interrogative Pronouns (Vragende Voornaamwoorden)

| Dutch | English | Usage |
|-------|---------|-------|
| wie | who | people |
| wat | what | things |
| welke | which | de-words/plurals |
| welk | which | het-words |
| wiens | whose | possession |
| waar | where | place |
| wanneer | when | time |
| waarom | why | reason |
| hoe | how | manner |

### Examples
- **Wie** is dat? (Who is that?)
- **Wat** doe je? (What are you doing?)
- **Welke** film wil je zien? (Which film do you want to see?)
- **Welk** boek lees je? (Which book are you reading?)
- **Wiens** auto is dit? (Whose car is this?)

## Er (There/It)

"Er" has multiple functions in Dutch:

### 1. Existential (There is/are)
- **Er** is een probleem. (There is a problem.)
- **Er** zijn veel mensen. (There are many people.)

### 2. Locative (There - referring to place)
- Ik woon **er**. (I live there.)
- Ben je **er** geweest? (Have you been there?)

### 3. Prepositional (Er + preposition)
- Ik denk **eraan**. (I think about it.)
- Hij wacht **erop**. (He waits for it.)
- Waar denk je **aan**? → Ik denk **eraan**.

### 4. Partitive (Of them/of it)
- Hoeveel kinderen heb je? Ik heb **er** drie. (I have three of them.)
- Wil je koffie? Ja, ik wil **er** graag een. (I'd like one.)

### 5. Passive subject placeholder
- **Er** wordt gewerkt. (Work is being done.)

## Practice Exercises

### Exercise 1: Fill in the object pronoun
1. Ik zie Maria. Ik zie _______.
2. Ken je mijn ouders? Ja, ik ken _______.
3. Waar is het boek? Ik heb _______ niet gezien.
4. Dit cadeau is voor jou. Dit is voor _______.
5. Wij zijn hier. Kun je _______ helpen?

### Exercise 2: Use the correct possessive
1. Dit is _______ (ik) auto.
2. Waar zijn _______ (jullie) kinderen?
3. _______ (wij) huis is groot. (het-word)
4. Ik ken _______ (hij) broer.
5. _______ (zij - she) moeder is aardig.

### Exercise 3: Choose dit/dat/deze/die
1. _______ boek is interessant. (this, het-word)
2. _______ schoenen zijn van mij. (those)
3. _______ man ken ik niet. (that)
4. _______ is mijn favoriet. (this)
5. _______ huizen zijn duur. (these)

### Answers

**Exercise 1:** 1. haar, 2. hen/ze, 3. het, 4. jou, 5. ons

**Exercise 2:** 1. mijn, 2. jullie, 3. Ons, 4. zijn, 5. Haar

**Exercise 3:** 1. Dit, 2. Die, 3. Die, 4. Dit, 5. Deze

## Key Points to Remember

✅ **Je/ze** are unstressed, **jij/zij** are stressed  
✅ **Ons** (het-words) vs **onze** (de-words/plurals)  
✅ **Deze/dit** (near) vs **die/dat** (far)  
✅ **Reflexive zich** for hij/zij/het/zij (plural)  
✅ **Er** has 5 functions: existential, locative, prepositional, partitive, passive  
✅ **Relative pronouns**: die (de-words), dat (het-words)

# A2 Dutch: Daily Conversations (Dagelijkse Gesprekken)

## Overview
This document provides common conversation patterns and dialogues for everyday situations.

---

## 1. Introducing Yourself (Jezelf Voorstellen)

### Basic Introduction
**A:** Hallo! Ik ben Anna. Hoe heet jij?  
**B:** Hoi! Ik heet Peter. Leuk je te ontmoeten!  
**A:** Insgelijks! Waar kom je vandaan?  
**B:** Ik kom uit Duitsland, maar ik woon nu in Amsterdam. En jij?  
**A:** Ik kom uit Nederland. Ik ben geboren in Rotterdam.

### Key Phrases
| Dutch | English |
|-------|---------|
| Ik ben... | I am... |
| Ik heet... | My name is... |
| Aangenaam! | Pleased to meet you! |
| Leuk je te ontmoeten! | Nice to meet you! |
| Insgelijks! | Likewise! |
| Waar kom je vandaan? | Where are you from? |
| Ik kom uit... | I come from... |
| Ik woon in... | I live in... |
| Wat doe je (voor werk)? | What do you do (for work)? |
| Ik ben student/leraar/arts | I am a student/teacher/doctor |
| Ik werk bij... | I work at... |

## 2. At a Shop (In de Winkel)

### Buying Groceries
**Verkoper:** Goedemiddag! Kan ik u helpen?  
**Klant:** Ja, ik zoek kaas. Heeft u oude kaas?  
**Verkoper:** Natuurlijk! Deze is erg lekker. Hoeveel wilt u?  
**Klant:** 200 gram alstublieft.  
**Verkoper:** Alstublieft. Wilt u nog iets anders?  
**Klant:** Nee, dank u. Dat is alles.  
**Verkoper:** Dat is dan 4,50 euro.  
**Klant:** Kan ik pinnen?  
**Verkoper:** Ja hoor. Alstublieft, de bon.  
**Klant:** Dank u wel. Tot ziens!

### Buying Clothes
**Klant:** Mag ik dit shirt passen?  
**Verkoper:** Natuurlijk! De paskamers zijn daar.  
**Klant:** Heeft u dit ook in een grotere maat?  
**Verkoper:** Ik zal even kijken... Ja, hier in maat L.  
**Klant:** Perfect, ik neem hem. Hoeveel kost het?  
**Verkoper:** Dit shirt kost 29,99 euro.  
**Klant:** Is er korting?  
**Verkoper:** Ja, 20% korting. Het wordt 23,99 euro.

### Key Phrases
| Dutch | English |
|-------|---------|
| Kan ik u helpen? | Can I help you? |
| Ik zoek... | I'm looking for... |
| Hoeveel kost dit/het? | How much does this cost? |
| Wat is de prijs? | What is the price? |
| Dat is te duur. | That's too expensive. |
| Heeft u korting? | Do you have a discount? |
| Heeft u dit in een andere kleur/maat? | Do you have this in another color/size? |
| Mag ik dit passen? | May I try this on? |
| Ik neem het. | I'll take it. |
| Alleen kijken, dank u. | Just looking, thanks. |
| Kan ik pinnen? | Can I pay by card? |
| Contant betalen | To pay cash |
| De bon | Receipt |

## 3. At a Restaurant (In het Restaurant)

### Ordering Food
**Ober:** Goedenavond! Heeft u gereserveerd?  
**Gast:** Ja, op naam van De Vries, voor twee personen.  
**Ober:** Welkom! Komt u maar mee. Dit is uw tafel.  
**Gast:** Dank u. Mogen we de menukaart?  
**Ober:** Natuurlijk. Wilt u alvast iets drinken?  
**Gast:** Twee glazen witte wijn, alstublieft.  
**Ober:** Prima. Heeft u al een keuze gemaakt?  
**Gast:** Ja, ik wil graag de tomatensoep als voorgerecht en de zalm als hoofdgerecht.  
**Ober:** Uitstekende keuze. En voor u?  
**Gast 2:** Voor mij de salade en de biefstuk, medium gebakken alstublieft.  
**Ober:** Komt eraan!

### After the Meal
**Gast:** Mag ik de rekening, alstublieft?  
**Ober:** Natuurlijk. Was alles naar wens?  
**Gast:** Ja, het was heerlijk! Complimenten aan de kok.  
**Ober:** Dank u wel! De rekening komt eraan.

### Key Phrases
| Dutch | English |
|-------|---------|
| Een tafel voor twee, alstublieft. | A table for two, please. |
| Heeft u gereserveerd? | Did you make a reservation? |
| De menukaart, alstublieft. | The menu, please. |
| Wat is de dagschotel? | What's the dish of the day? |
| Ik wil graag bestellen. | I would like to order. |
| Als voorgerecht / hoofdgerecht / dessert | As appetizer / main course / dessert |
| Ik neem de... | I'll have the... |
| Hoe is de ...? | How is the...? |
| Ik ben vegetariër/veganist. | I'm vegetarian/vegan. |
| Ik heb een allergie voor... | I have an allergy to... |
| De rekening, alstublieft. | The bill, please. |
| Inclusief fooi | Including tip |
| Eet smakelijk! | Enjoy your meal! |
| Het was heerlijk! | It was delicious! |

## 4. Asking for Directions (De Weg Vragen)

### Finding a Place
**Persoon A:** Pardon, kunt u mij helpen? Ik zoek het station.  
**Persoon B:** Natuurlijk! Loop rechtdoor tot het verkeerslicht.  
**Persoon A:** Oké, rechtdoor.  
**Persoon B:** Dan sla je linksaf. Het station is aan je rechterhand.  
**Persoon A:** Hoe ver is het?  
**Persoon B:** Ongeveer vijf minuten lopen.  
**Persoon A:** Hartelijk dank!  
**Persoon B:** Graag gedaan!

### Key Phrases
| Dutch | English |
|-------|---------|
| Pardon, kunt u mij helpen? | Excuse me, can you help me? |
| Ik zoek... | I'm looking for... |
| Hoe kom ik bij...? | How do I get to...? |
| Is het ver? | Is it far? |
| Ga rechtdoor | Go straight |
| Sla linksaf | Turn left |
| Sla rechtsaf | Turn right |
| Op de hoek | On the corner |
| Aan je linkerhand/rechterhand | On your left/right |
| Bij het verkeerslicht | At the traffic light |
| Bij de rotonde | At the roundabout |
| De eerste/tweede straat links | The first/second street on the left |
| Tegenover de... | Opposite the... |
| Naast de... | Next to the... |
| Het is om de hoek | It's around the corner |

## 5. Making Appointments (Afspraken Maken)

### Doctor's Appointment
**Receptionist:** Goedemorgen, huisartsenpraktijk De Berg.  
**Patiënt:** Goedemorgen. Ik wil graag een afspraak maken.  
**Receptionist:** Wat zijn uw klachten?  
**Patiënt:** Ik heb al drie dagen hoofdpijn en koorts.  
**Receptionist:** Dat klinkt vervelend. Kunt u morgenochtend om tien uur?  
**Patiënt:** Ja, dat past prima.  
**Receptionist:** Mag ik uw naam en geboortedatum?  
**Patiënt:** Jan Bakker, 15 maart 1985.  
**Receptionist:** Prima, u staat genoteerd voor morgen 10 uur.

### Key Phrases
| Dutch | English |
|-------|---------|
| Ik wil graag een afspraak maken. | I would like to make an appointment. |
| Wanneer kunt u? | When are you available? |
| Schikt het om...? | Does ... work for you? |
| Heeft u tijd op...? | Do you have time on...? |
| Dat past mij goed. | That works for me. |
| Dat komt mij niet uit. | That doesn't work for me. |
| Kunnen we verzetten naar...? | Can we reschedule to...? |
| Ik moet de afspraak annuleren. | I have to cancel the appointment. |
| Tot dan! | See you then! |

## 6. At the Doctor (Bij de Dokter)

### Describing Symptoms
**Dokter:** Goedemiddag, wat kan ik voor u doen?  
**Patiënt:** Ik voel me niet lekker. Ik heb keelpijn en ik ben erg moe.  
**Dokter:** Hoe lang heeft u deze klachten al?  
**Patiënt:** Sinds drie dagen.  
**Dokter:** Heeft u ook koorts?  
**Patiënt:** Ja, 38,5 graden.  
**Dokter:** Ik ga even kijken. Kunt u uw mond openen?  
**Patiënt:** Natuurlijk.  
**Dokter:** Het lijkt op een keelontsteking. Ik schrijf antibiotica voor.  
**Patiënt:** Hoeveel tabletten per dag?  
**Dokter:** Drie keer per dag één tablet, na het eten.

### Key Phrases
| Dutch | English |
|-------|---------|
| Ik voel me niet lekker. | I don't feel well. |
| Ik heb pijn in mijn... | I have pain in my... |
| Ik heb hoofdpijn/buikpijn/rugpijn. | I have a headache/stomachache/backache. |
| Ik heb koorts. | I have a fever. |
| Ik ben verkouden. | I have a cold. |
| Ik hoest veel. | I cough a lot. |
| Ik heb griep. | I have the flu. |
| Ik ben duizelig. | I am dizzy. |
| Ik ben misselijk. | I feel nauseous. |
| Ik ben allergisch voor... | I am allergic to... |
| Ik neem medicijnen. | I take medication. |
| Kunt u iets voorschrijven? | Can you prescribe something? |
| Het recept | The prescription |

## 7. Public Transport (Openbaar Vervoer)

### Buying a Ticket
**Reiziger:** Goedemorgen! Een enkele reis naar Utrecht, alstublieft.  
**Loket:** Eerste of tweede klas?  
**Reiziger:** Tweede klas.  
**Loket:** Dat is 11,50 euro.  
**Reiziger:** Van welk spoor vertrekt de trein?  
**Loket:** Spoor 4. De trein vertrekt over tien minuten.  
**Reiziger:** Moet ik overstappen?  
**Loket:** Nee, dit is een directe verbinding.

### Key Phrases
| Dutch | English |
|-------|---------|
| Een enkele reis naar... | A single ticket to... |
| Een retourtje naar... | A return ticket to... |
| Eerste/tweede klas | First/second class |
| Hoe laat vertrekt de trein? | What time does the train leave? |
| Van welk spoor/perron? | From which platform? |
| Waar moet ik overstappen? | Where do I transfer? |
| Is dit een directe verbinding? | Is this a direct connection? |
| De trein heeft vertraging. | The train is delayed. |
| De trein is geannuleerd. | The train is cancelled. |
| Hoe lang duurt de reis? | How long does the journey take? |
| Is deze plaats vrij? | Is this seat free? |

## 8. Making Plans (Plannen Maken)

### Inviting Someone
**A:** Hoi! Heb je zaterdag iets te doen?  
**B:** Nee, ik heb nog geen plannen. Hoezo?  
**A:** Zullen we naar de bioscoop gaan?  
**B:** Leuk idee! Welke film wil je zien?  
**A:** Die nieuwe actiefilm. Heb je zin?  
**B:** Ja, lijkt me leuk! Hoe laat spreken we af?  
**A:** Zullen we om zeven uur afspreken bij de bioscoop?  
**B:** Prima! Tot zaterdag!

### Key Phrases
| Dutch | English |
|-------|---------|
| Heb je zin om...? | Do you feel like...? |
| Zullen we...? | Shall we...? |
| Heb je tijd om...? | Do you have time to...? |
| Wat vind je van...? | What do you think of...? |
| Lijkt me leuk! | Sounds fun! |
| Goed idee! | Good idea! |
| Ik heb geen zin. | I don't feel like it. |
| Sorry, ik kan niet. | Sorry, I can't. |
| Ik heb al een afspraak. | I already have an appointment. |
| Waar spreken we af? | Where shall we meet? |
| Hoe laat spreken we af? | What time shall we meet? |

## 9. Phone Conversations (Telefoongesprekken)

### Calling Someone
**A:** Hallo, met Peter.  
**B:** Hoi Peter, met Anna. Stoor ik?  
**A:** Nee hoor, zeg het maar.  
**B:** Ik bel over de vergadering van morgen. Hoe laat begint die?  
**A:** Om twee uur. Kun je komen?  
**B:** Ja, ik ben erbij. Tot morgen!  
**A:** Tot dan, doei!

### Key Phrases
| Dutch | English |
|-------|---------|
| Hallo, met... | Hello, this is... |
| Met wie spreek ik? | Who am I speaking with? |
| Kan ik ... spreken? | Can I speak to...? |
| Ogenblik alstublieft. | One moment please. |
| Sorry, verkeerd verbonden. | Sorry, wrong number. |
| Kunt u dat herhalen? | Can you repeat that? |
| Ik bel later terug. | I'll call back later. |
| Kunt u een boodschap achterlaten? | Can you leave a message? |
| Ik stuur je een berichtje. | I'll send you a text. |

## 10. Small Talk (Kletsen)

### About the Weather
**A:** Wat een lekker weer vandaag!  
**B:** Ja hè! Eindelijk zon na al die regen.  
**A:** Zullen we buiten gaan zitten?  
**B:** Goed idee! Maar morgen gaat het weer regenen.  
**A:** Typisch Nederlands weer...

### About Weekend Plans
**A:** Heb je een leuk weekend gehad?  
**B:** Ja, heel gezellig! We zijn naar de markt geweest.  
**A:** Oh leuk! En verder nog iets gedaan?  
**B:** Zondag zijn we thuis gebleven. Lekker rustig.  
**A:** Klinkt goed!

### Key Phrases for Small Talk
| Dutch | English |
|-------|---------|
| Hoe gaat het met je? | How are you? |
| Goed hoor, en met jou? | Fine, and you? |
| Alles goed? | Everything okay? |
| Wat een weer! | What weather! |
| Gezellig! | Cozy/Nice! |
| Wat leuk! | How nice! |
| Jammer! | What a pity! |
| Echt waar? | Really? |
| Dat meen je niet! | You're kidding! |
| Klopt! | That's right! |

## Key Points to Remember

✅ Always start with a greeting  
✅ Use "u" for formal situations, "je/jij" for informal  
✅ "Alstublieft" (formal) vs "Alsjeblieft" (informal)  
✅ Practice common question patterns  
✅ Learn fixed expressions for each situation

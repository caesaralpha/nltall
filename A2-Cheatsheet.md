# A2 Dutch Cheatsheet (Spiekbriefje)

## Quick Reference Guide for A2 Level

---

## VERB CONJUGATION

### Regular Present Tense
| Pronoun | Ending | Example: werken |
|---------|--------|-----------------|
| ik | stem | werk |
| jij/u/hij/zij | stem + t | werkt |
| wij/jullie/zij | infinitive | werken |

**Inversion**: Werk **jij**? (drop -t when jij follows verb)

### Irregular Verbs - MEMORIZE!
| | zijn | hebben | gaan | doen | kunnen | willen |
|---|------|--------|------|------|--------|--------|
| ik | ben | heb | ga | doe | kan | wil |
| jij | bent | hebt | gaat | doet | kunt | wilt |
| hij | is | heeft | gaat | doet | kan | wil |
| wij | zijn | hebben | gaan | doen | kunnen | willen |

### Past Tense ('t kofschip)
Stem ends in **t, k, f, s, ch, p** → add **-te/-ten**  
Otherwise → add **-de/-den**

| | 't kofschip | Not 't kofschip |
|---|-------------|-----------------|
| Singular | werkte | woonde |
| Plural | werkten | woonden |

### Perfect Tense
**Hebben/Zijn + Past Participle (at end)**

**ZIJN** with: beweging (gaan, komen, rijden) + verandering (worden) + zijn/blijven

| Type | Formula | Example |
|------|---------|---------|
| Regular | ge + stem + t/d | gewerkt, gewoond |
| Strong | ge + changed + en | geschreven, gelezen |
| Separable | prefix + ge + stem | opgebeld, aangekomen |
| be/ver/ont | no ge- | bezocht, vergeten |

## WORD ORDER

### Main Clause: V2 RULE
Verb ALWAYS position 2!

| Pos 1 | Pos 2 (VERB) | Subject | Rest | Final Verb |
|-------|--------------|---------|------|------------|
| Ik | werk | - | morgen | - |
| Morgen | werk | ik | - | - |
| Ik | heb | - | hard | gewerkt |
| Morgen | zal | ik | - | komen |

### Time - Manner - Place
Ik ga **morgen** (T) **met de trein** (M) **naar Amsterdam** (P).

### Negation
- **niet**: before adjective, final verb, place → Ik kan **niet** zwemmen.
- **geen**: replaces een → Ik heb **geen** tijd.

## ADJECTIVES

### -e Endings
| | de-word | het + definite | het + een |
|---|---------|----------------|-----------|
| Add -e? | ✓ grote | ✓ grote | ✗ groot |

**Never -e**: predicates (Het huis is groot), -en adjectives (open, houten)

### Spelling Changes
- groot → grot**e** (not groote)
- dik → dikk**e**
- lief → liev**e** (f→v)

---

## ARTICLES (de/het)

### DE Words
- Plurals (always!)
- People, professions
- -ie, -ij, -heid, -nis, -ing, -tie, -teit, -schap

### HET Words
- Diminutives (-je): het meisje
- Infinitives as nouns: het eten
- Languages: het Nederlands
- ge- words: het gebouw
- -um, -ment, -sel, -isme

### Compound Rule
Last word decides: de slaap**kamer** (de kamer)

## PREPOSITIONS

### Location
| Dutch | When |
|-------|------|
| in | enclosed space: in de stad |
| op | surface, institution: op tafel, op school |
| aan | attached: aan de muur |
| bij | near: bij het station |
| naar | motion to: naar Amsterdam |
| uit | from: uit Nederland |

### Time
| Dutch | When |
|-------|------|
| om | clock time: om drie uur |
| op | day/date: op maandag |
| in | month/year: in januari |

### Verb + Preposition (MEMORIZE)
- denken **aan**
- wachten **op**
- luisteren **naar**
- houden **van**
- praten **over**

## MODAL VERBS

| Modal | Meaning | Past |
|-------|---------|------|
| kunnen | can | kon |
| mogen | may | mocht |
| moeten | must | moest |
| willen | want | wilde |
| zullen | will | zou |
| hoeven | need (neg) | hoefde |

Structure: Modal (V2) + ... + Infinitive (end)
- Ik **kan** Nederlands **spreken**.
- **Hoeven** + **te** + infinitive!

## NUMBERS & TIME

### Time Expressions
- **Half** = half TO next hour: half drie = 2:30
- Kwart over drie = 3:15
- Kwart voor drie = 2:45
- Tien over drie = 3:10
- Tien voor drie = 2:50

### Dates
- Op 5 mei (on May 5th)
- In januari (in January)
- In 2024 (in 2024)

## COMMON MISTAKES TO AVOID

❌ Ik **bent** → ✅ Ik **ben**  
❌ Hij **werke** → ✅ Hij **werkt**  
❌ Een groot**e** huis → ✅ Een groot huis (het-word + een)  
❌ Half drie = 3:30 → ✅ Half drie = **2:30**  
❌ Ik heb gegaan → ✅ Ik **ben** gegaan  
❌ Ik hoef niet komen → ✅ Ik hoef niet **te** komen

# B1 Dutch: The Many Uses of "Er"

## Overview
"Er" is one of the trickiest words in Dutch. It has multiple functions that don't translate directly into English. Mastering "er" is essential for B1 level fluency.

---

## Five Uses of "Er"

| Type | Function | Example |
|------|----------|---------|
| 1. Locative | there (place) | Ik ben **er** geweest. |
| 2. Existential | there (existence) | **Er** zijn veel mensen. |
| 3. Prepositional | replaces prep + het | Ik denk **eraan**. |
| 4. Partitive | some/of them | Ik heb **er** drie. |
| 5. Passive | indefinite subject | **Er** wordt gedanst. |

---

## 1. Locative "Er" (Place)

### Function
Refers to a place that was mentioned before. Similar to "there" in English.

### Examples
- Ken je Amsterdam? Ja, ik heb **er** gewoond.
  (Do you know Amsterdam? Yes, I lived **there**.)
- Het restaurant is goed. Ik eet **er** vaak.
  (The restaurant is good. I often eat **there**.)
- Is Jan op kantoor? Nee, hij is **er** niet.
  (Is Jan at the office? No, he's not **there**.)

### Note
Can be replaced by "daar" for emphasis:
- Ik heb **daar** gewoond. (I lived **there** [specifically].)

---

## 2. Existential "Er" (There is/are)

### Function
Introduces existence of something. Like "there is/there are" in English.

### Structure
**Er + zijn + subject**

### Examples
- **Er** is een probleem. (There is a problem.)
- **Er** zijn drie kinderen in de klas. (There are three children in class.)
- **Er** was niemand thuis. (There was nobody home.)
- **Er** waren veel mensen op het feest. (There were many people at the party.)

### Word Order
"Er" takes first position but can be omitted after inversion:
- **Er** zijn veel mensen. 
- Gisteren waren **(er)** veel mensen. (er is optional here)
- **(Er)** Zijn hier veel mensen? (er is optional in questions)

---

## 3. Prepositional "Er" (Er + Preposition)

### Function
Replaces a preposition + "het" (things/concepts). Cannot be used for people!

### How it Works
| Instead of | Use |
|------------|-----|
| op het | er**op** |
| aan het | er**aan** |
| voor het | er**voor** |
| met het | er**mee** |
| over het | er**over** |
| naar het | er**naar** |

**Note:** "Met" becomes "mee"

### Examples

#### Thinking about it
- ❌ Ik denk aan het (incorrect)
- ✅ Ik denk **eraan**. (I think about it.)

#### Looking at it
- ❌ Kijk naar het!
- ✅ Kijk **ernaar**!

#### Working on it
- Ik werk **eraan**. (I'm working on it.)

#### Talking about it
- We praten **erover**. (We talk about it.)

#### Using it
- Ik doe niets **ermee**. (I don't do anything with it.)

---

### Splitting Er + Preposition

In main clauses, "er" and the preposition often split:
- Ik denk **er** vaak **aan**. (I often think about it.)
- Zij praat **er** nooit **over**. (She never talks about it.)
- Hij houdt **er** niet **van**. (He doesn't like it.)

In subordinate clauses, they can stay together:
- ...omdat ik **eraan** denk.
- ...dat hij **ervan** houdt.

### With Specific Things (Daar/Hier/Waar)

| With | Meaning |
|------|---------|
| daar + aan = **daaraan** | about that |
| hier + mee = **hiermee** | with this |
| waar + over = **waarover** | about what |

Examples:
- **Daaraan** heb ik niet gedacht. (I didn't think about that.)
- **Waarmee** kan ik je helpen? (How can I help you? / With what...)
- **Hierover** wil ik praten. (I want to talk about this.)

---

## 4. Partitive "Er" (Some/Of them)

### Function
Refers to a quantity of something mentioned before. Often used with numbers.

### Structure
**Er + verb + number/quantity**

### Examples

**Question:** Hoeveel kinderen heb je?  
**Answer:** Ik heb **er** drie. (I have three [of them].)

**Question:** Heb je appels?  
**Answer:** Ja, ik heb **er** een paar. (Yes, I have a few [of them].)

**Question:** Zijn er stoelen?  
**Answer:** Ja, **er** zijn **er** vijf. (Yes, there are five [of them].)

### More Examples
- Hoeveel boeken heb je? – Ik heb **er** veel.
- Ik hou van bloemen. Ik heb **er** altijd in huis.
- We hadden tien stoelen, maar nu hebben we **er** acht.

---

## 5. Passive/Impersonal "Er"

### Function
Used as a dummy subject in passive sentences without a clear agent.

### Examples
- **Er** wordt hier niet gerookt. (No smoking here. / One doesn't smoke here.)
- **Er** wordt veel gepraat. (There's a lot of talking.)
- **Er** werd hard gewerkt. (Hard work was done.)
- **Er** wordt gedanst. (There is dancing. / People are dancing.)

---

## Combining Multiple Uses

Sometimes sentences have more than one "er":

### Existential + Partitive
- **Er** zijn **er** drie. (There are three of them.)
  - First "er" = existential (there)
  - Second "er" = partitive (of them)

### Locative + Prepositional
- Ik ben **er** geweest en heb **er** veel **over** gehoord.
  - First "er" = locative (there)
  - Second "er...over" = prepositional (about it)

---

## Position of "Er" in Sentences

### Main Clause (Standard)
"Er" comes after the conjugated verb:
- Ik denk **er** vaak aan.
- Hij houdt **er** niet van.

### After Inversion
"Er" comes after the subject:
- Morgen ga ik **er** naartoe.
- Vaak denk ik **er** aan.

### Subordinate Clause
"Er" comes early, before other elements:
- ...dat ik **er** vaak aan denk.
- ...omdat hij **er** niet van houdt.

## Common Expressions with "Er"

| Dutch | English |
|-------|---------|
| Hoe gaat het ermee? | How's it going? |
| Er is niets aan de hand. | Nothing's wrong. |
| Ik kan er niets aan doen. | I can't help it. |
| Daar heb je het al. | There you go (already). |
| Er komt niets van. | Nothing will come of it. |
| Wat moet ik ermee? | What should I do with it? |
| Ik heb er genoeg van. | I've had enough of it. |
| Ik ben er klaar mee. | I'm done with it. |
| Er valt niets aan te doen. | Nothing can be done about it. |
| Er zit wat in. | There's something to it. |

---

## Er vs Daar vs Hier

| Word | Use |
|------|-----|
| Er | Unstressed, neutral |
| Daar | Stressed, emphasizes "there" |
| Hier | Stressed, emphasizes "here" |

### Examples
- Ik denk **er**aan. (neutral)
- **Daar**aan denk ik vaak! (emphasis on "that")
- Ik hou **hier**van! (emphasis on "this")

## Key Points to Remember

✅ **5 types:** Locative, Existential, Prepositional, Partitive, Passive  
✅ **Prepositional:** Replaces preposition + het (things only, not people!)  
✅ **Partitive:** Used with numbers/quantities  
✅ **Position:** After conjugated verb in main clause  
✅ **Er + preposition** can split in main clauses  
✅ **"met" → "mee"** (ermee, daarmee, waarmee)  
✅ "Er" can occur twice in one sentence!

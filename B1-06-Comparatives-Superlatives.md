# B1 Dutch: Comparatives and Superlatives (Vergrotende en Overtreffende Trap)

## Overview
Comparatives compare two things; superlatives express the highest degree. Dutch forms these similarly to English but with some important differences.

---

## Regular Comparatives

### Formation
**Adjective + -er**

| Base | Comparative | English |
|------|-------------|---------|
| groot | groter | bigger |
| klein | kleiner | smaller |
| lang | langer | longer/taller |
| kort | korter | shorter |
| oud | ouder | older |
| jong | jonger | younger |
| snel | sneller | faster |
| langzaam | langzamer | slower |
| mooi | mooier | more beautiful |
| lelijk | lelijker | uglier |
| duur | duurder | more expensive |
| goedkoop | goedkoper | cheaper |

---

## Regular Superlatives

### Formation
**Adjective + -st(e)**

Use **-st** in predicative position (after verb)  
Use **-ste** in attributive position (before noun)

| Base | Superlative (pred) | Superlative (attr) |
|------|-------------------|-------------------|
| groot | het grootst | de/het grootste |
| klein | het kleinst | de/het kleinste |
| mooi | het mooist | de/het mooiste |
| snel | het snelst | de/het snelste |

### Examples
**Predicative (after verb):**
- Dit huis is **het grootst**.
- Deze auto is **het snelst**.
- Zij is **het mooist**.

**Attributive (before noun):**
- Dit is het **grootste** huis.
- Hij is de **snelste** loper.
- Zij is de **mooiste** vrouw.

## Spelling Rules

### 1. Short vowel + consonant → double consonant
| Base | Comparative | Superlative |
|------|-------------|-------------|
| dik | dikker | dikst(e) |
| dun | dunner | dunst(e) |
| nat | natter | natst(e) |
| wit | witter | witst(e) |

### 2. Long vowel → stays single
| Base | Comparative | Superlative |
|------|-------------|-------------|
| groot | groter | grootst(e) |
| hoog | hoger | hoogst(e) |
| laag | lager | laagst(e) |

### 3. -r ending → add -der (comparative)
| Base | Comparative | Superlative |
|------|-------------|-------------|
| duur | duurder | duurst(e) |
| ver | verder | verst(e) |
| zwaar | zwaarder | zwaarst(e) |

### 4. -s ending → add -t only (superlative)
| Base | Comparative | Superlative |
|------|-------------|-------------|
| fris | frisser | frist(e) |
| boos | bozer | boost (no extra s) |

### 5. Words ending in -st, -sk, -isch → use "meest"
| Base | Comparative | Superlative |
|------|-------------|-------------|
| juist | juister | het meest juist |
| logisch | logischer | het meest logisch |

## Irregular Comparatives and Superlatives

### Must Memorize!
| Base | Comparative | Superlative | English |
|------|-------------|-------------|---------|
| goed | beter | best(e) | good/better/best |
| veel | meer | meest(e) | much/more/most |
| weinig | minder | minst(e) | little/less/least |
| graag | liever | liefst(e) | gladly/rather/most |

### Examples
- Dit boek is **beter** dan dat boek.
- Hij is de **beste** student in de klas.
- Ik heb **meer** tijd dan gisteren.
- Zij heeft het **meeste** geld.
- Ik drink **liever** thee dan koffie.
- Ik zwem **het liefst** in de zee.

## Comparisons with "dan" and "als"

### Dan (than) - Standard Dutch
- Hij is groter **dan** ik.
- Deze film is beter **dan** die.
- Ik heb meer geld **dan** jij.

### Als (than) - Spoken/Regional
Often heard in spoken Dutch, but "dan" is grammatically correct:
- Hij is groter **als** ik. (spoken)
- Better: Hij is groter **dan** ik. (standard)

### Net zo ... als (as ... as)
For equality:
- Hij is **net zo** groot **als** ik.
- Deze auto is **even** duur **als** die.
- Zij zingt **net zo** goed **als** een professional.

### Even ... als (just as ... as)
- Dit is **even** belangrijk **als** dat.
- Hij is **even** slim **als** zijn zus.

## Structures with Comparatives and Superlatives

### Comparative Structures
| Structure | Example |
|-----------|---------|
| [comp] dan | groter dan |
| steeds [comp] | steeds groter (bigger and bigger) |
| hoe [comp], hoe [comp] | hoe meer, hoe beter |
| [comp] en [comp] | groter en groter |
| veel/een stuk/iets [comp] | veel groter, iets kleiner |
| niet [comp] dan | niet groter dan (not bigger than) |

### Examples
- Het wordt **steeds kouder**. (It's getting colder and colder.)
- **Hoe meer** je oefent, **hoe beter** je wordt.
- De prijzen worden **hoger en hoger**.
- Dit huis is **veel groter** dan dat.
- Hij is **een stuk ouder** dan zij.
- Dit is **niets bijzonders** - "iets" can modify adjectives
- Het is **iets warmer** vandaag.

### Superlative Structures
- de/het [superlative] + noun
- het [superlative] (predicative)
- een van de [superlative] (one of the...)

### Examples
- Dit is **de hoogste** berg van Nederland.
- Hij is **het snelst**. (He is the fastest.)
- Zij is **een van de beste** studenten.
- Amsterdam is **een van de mooiste** steden.

---

## Absolute Comparatives

Some comparatives are used without comparison (absolute):
- de **oudere** generatie (the older generation)
- **hogere** opleiding (higher education)
- de **betere** kringen (the better circles)
- **mindere** kwaliteit (lesser quality)

---

## Using "meer" and "meest"

### For Long Adjectives
Some adjectives (especially those of foreign origin or with 3+ syllables) prefer "meer/meest":

| Base | Comparative | Superlative |
|------|-------------|-------------|
| interessant | interessanter / meer interessant | interessantst / meest interessant |
| comfortabel | comfortabeler / meer comfortabel | meest comfortabel |
| fantastisch | meer fantastisch | meest fantastisch |

### Adjectives Always with meer/meest
- tevreden → meer tevreden, meest tevreden
- gestrest → meer gestrest
- bezorgd → meer bezorgd

## Common Expressions

### With Comparatives
| Dutch | English |
|-------|---------|
| hoe sneller, hoe beter | the faster, the better |
| des te beter | all the better |
| beter laat dan nooit | better late than never |
| minder is meer | less is more |
| dat kan beter | that could be better |
| niets meer dan | nothing more than |

### With Superlatives
| Dutch | English |
|-------|---------|
| op zijn best | at his best |
| op zijn minst | at least |
| ten minste | at least (formal) |
| hoogstens | at most |
| uiterst | extremely |
| in het beste geval | in the best case |
| in het slechtste geval | in the worst case |

---

## Diminutives and Comparison

Diminutives can't be compared normally:
- ❌ huisjer, huisjest
- ✅ Use "meer" or describe differently

## Key Points to Remember

✅ Comparative: adjective + **-er**  
✅ Superlative: adjective + **-st(e)**  
✅ Irregular: goed/beter/best, veel/meer/meest, weinig/minder/minst  
✅ Use **dan** for comparisons (not als in standard Dutch)  
✅ Use **net zo ... als** for equality  
✅ Use **het** with superlatives (het grootst, het beste)  
✅ Double consonant for short vowels (dik → dikker)  
✅ **Hoe ... hoe ...** = the ... the ...

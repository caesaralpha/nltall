# A2 Dutch: Present Tense Verbs (Tegenwoordige Tijd)

## Overview
The present tense in Dutch is used to describe current actions, habits, general truths, and future events with a time indicator.

---

## Regular Verb Conjugation

### Pattern: Remove -en from infinitive, add personal endings

| Pronoun | Ending | Example: werken (to work) |
|---------|--------|---------------------------|
| ik | stem | ik werk |
| jij/je | stem + t | jij werkt |
| u | stem + t | u werkt |
| hij/zij/het | stem + t | hij werkt |
| wij/we | infinitive | wij werken |
| jullie | infinitive | jullie werken |
| zij | infinitive | zij werken |

### Important Rule: Inversion with "jij/je"
When the verb comes before "jij/je", drop the -t:
- **Jij werkt** hier. (You work here.)
- **Werk jij** hier? (Do you work here?) ← No -t!

---

## Spelling Rules for Stems

### 1. Double Vowel Rule
If the infinitive has a double vowel (aa, ee, oo, uu) in a closed syllable, keep single vowel in open syllable:
- **lopen** (to walk) → ik **loop** (closed syllable needs double o)
- **maken** (to make) → ik **maak**

### 2. Single Vowel Rule
Single vowels in open syllables stay single:
- **leven** (to live) → ik **leef** (v→f at end)

### 3. Consonant Changes
- **v → f** at the end: schrijven → ik schrijf
- **z → s** at the end: reizen → ik reis

### 4. Double Consonant Rule
Double consonants become single when at the end:
- **zitten** (to sit) → ik **zit**
- **rennen** (to run) → ik **ren**

## Practice Exercises

### Exercise 1: Fill in the correct verb form
1. Ik _______ (werken) in een kantoor.
2. Hij _______ (wonen) in Amsterdam.
3. _______ jij (komen) morgen?
4. Wij _______ (zijn) studenten.
5. Zij _______ (hebben) twee kinderen.

### Exercise 2: Correct the mistakes
1. Ik bent moe.
2. Hij werke hard.
3. Wij gaat naar school.
4. Jij bent thuis?

### Answers
**Exercise 1:** 1. werk, 2. woont, 3. Kom, 4. zijn, 5. hebben
**Exercise 2:** 1. ben, 2. werkt, 3. gaan, 4. Ben jij

---

## Key Points to Remember

✅ **Stem + t** for jij, u, hij/zij/het  
✅ **Infinitive** for wij, jullie, zij  
✅ **Inversion**: drop -t when verb comes before jij  
✅ **Spelling changes**: v→f, z→s at word end  
✅ **Memorize irregular verbs**: zijn, hebben, gaan, doen, komen
